CreateDeliveryTask 创建取件任务

功能：

创建取件任务

URL：http://后台/taskpool

method：CreateDeliveryTask

参数：

字段 选项 类型 长度 说明 选值 示例
uid 必选 String 32 用户ID - -
barcodes 必选 Array - 运单号列表 - -

barcodes:

每个单元：

字段 选项 类型 长度 说明 选值 示例
- 必选 String 32 运单号 - -

返回：

无

示例：

请求消息

{
    "jsonrpc": "2.0", 
    "id": "0123456789",
    "method": "CreateDeliveryTask",
    "params": {
        "nodeid": "xxx"
        "barcodes": [
                "xxx",
                "xxx"
        ]
    }, 
}

响应消息

{
    "jsonrpc": "2.0",
    "id": "0123456789",
    "result": {
        "code": "200",
        "message": "OK"
    }
}

GetDeliveryTask 巴枪从树莓派获取取件任务

功能：

巴枪从树莓派获取取件任务

URL：http://树莓派/taskpool

method：GetDeliveryTask

参数：

字段 选项 类型 长度 说明 选值 示例
nodeid 必选 String 32 站点ID - -
scannerid 必选 String 64 巴枪ID - -
staffid 必选 String 64 店员ID - -

返回：

字段 选项 类型 长度 说明 选值 示例
tasks 必选 Array - 任务列表 - -

说明：目前只实现一个任务，考虑以后扩展，使用了列表。

tasks:

每个单元如下：

字段 选项 类型 长度 说明 选值 示例
taskid 必选 Number - 任务ID - -
mobile 必选 String 16 手机 - -
nodeid 必选 String 32 站点ID - -
createtime 必选 String - 创建时间 - -
packages 必选 Array - 包裹列表 - -

packages:

字段 选项 类型 长度 说明 选值 示例
barcode 必选 String 64 运单号 - -
freight 必选 String/Number - 运费 - -
checkcode 必选 String/Number - 密码 - -
boxid 必选 String 32 箱号 - -
seqno 必选 String 8 序号 - -
express 必选 Object - 快递公司 - -
nodeid 必选 String 32 站点ID - -
lockerid 必选 String 32 柜子ID - -
cod 必选 String/Number - 代收货款 - -

express:

字段 选项 类型 长度 说明 选值 示例
name 必选 String - 快递公司名字 - "顺丰速运"

示例：

请求消息

{
    "jsonrpc": "2.0", 
    "id": "0123456789",
    "method": "GetDeliveryTask",
    "params": {
        "nodeid": "xxx",
        "scannerid": "xxx",
        "staffid": "xxx",
    }, 
}

响应消息

{
    "jsonrpc": "2.0",
    "id": "0123456789",
    "result": {
        "code": "200",
        "tasks": [
            {
                "taskid": 1234,
                "mobile": "12312345678",
                "nodeid": "N94F34L93795798",
                "createtime": "2015-10-20 18:17:18",
                "packages": [
                    {
                        "barcode": "919878211297", 
                        "freight": "0", 
                        "checkcode": "7630", 
                        "boxid": "010101", 
                        "mobile": "18505817258", 
                        "seqno": "1", 
                        "express": {
                            "name": "顺丰速运"
                        }, 
                        "lockerid": "L94F41L000004416", 
                        "cod": "0", 
                    }, {
                        ...
                    }
                ]
            }
        ]
    }
}
